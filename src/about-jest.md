# Jestとは

Jest公式サイト: [https://jestjs.io/ja](https://jestjs.io/ja)

Jestは、JavaScriptのテストを行うためのフレームワークです。
Facebookによってメンテナンスを続けられており、2022年現在でもJavaScriptを使用する多くの開発者が使用している人気のツールです。

>

少ない設定で簡単にテストをはじめることができます。
